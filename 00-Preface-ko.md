# 서문

2022년 말부터 ChatGPT로 대표되는 LLM(대형 언어 모델)은 기술적 쓰나미처럼 전 세계를 휩쓸며 인공 지능과 상호 작용하는 방식을 완전히 변화시켰습니다. LLM의 강력한 자연어 이해 및 생성 기능은 우리에게 인공 일반 지능(AGI)을 향한 길을 엿볼 수 있게 해주었습니다. 그러나 처음의 놀라움이 가라앉자 개발자들은 AI를 단순한 "질문 답변" 도구가 아니라 자율 계획, 도구 호출 및 복잡한 문제 해결이 가능한 "행위자"로 만드는 방법이라는 다음 개척지를 탐구하기 시작했습니다.

대답은 **에이전트**입니다.

2024년이 '백 모델 전쟁'의 원년이었다면, 2025년은 의심할 여지없이 '에이전트의 해'를 맞이한 해입니다. 우리는 기술적 초점이 더 크고 더 강력한 기초 모델을 훈련하는 것에서 더 스마트하고 효율적인 에이전트 애플리케이션을 구축하는 것으로 옮겨가고 있음을 알 수 있습니다. 개별 에이전트는 이미 특정 도메인의 작업을 처리할 수 있는 반면, 여러 에이전트가 노동 분업, 협력을 통해 협력하고 심지어 거대한 목표를 달성하기 위해 토론하는 다중 에이전트 시스템(MAS)은 LLM의 잠재력을 최대한 활용하고 복잡한 실제 문제를 해결하는 열쇠로 간주됩니다.

그러나 현재 생태계에는 명백한 격차가 있습니다. 한편으로는 어지러울 정도로 많은 에이전트 프레임워크와 애플리케이션이 지속적으로 등장하고 있습니다. 반면에 체계적인 지식은 극도로 부족합니다. 대부분의 튜토리얼은 특정 프레임워크에 대한 API 호출에 중점을 두므로 학습자는 "방법은 알지만 이유는 알 수 없습니다". 복잡한 요구 사항에 직면할 때 여전히 무력감을 느끼게 됩니다. 프레임워크 외관을 꿰뚫고 첫 번째 원칙부터 시작하여 에이전트 설계, 구성 및 협업을 체계적으로 설명할 수 있는 실용적인 가이드가 부족합니다.

이를 고려하여 우리는 커뮤니티에 에이전트 시스템을 처음부터 구축하고 이론과 실습의 균형을 맞추는 가이드를 제공하기 위해 Hello-Agents 프로젝트를 시작했습니다. 에이전트 분야의 최첨단 기술을 감상할 수 있도록 안내할 뿐만 아니라 핵심 아키텍처를 자세히 살펴보고 클래식 패러다임을 이해하며 궁극적으로 자신만의 멀티 에이전트 애플리케이션을 직접 구축할 수 있도록 안내합니다.

우리는 학습하는 가장 좋은 방법은 실습을 통해서라고 믿습니다. 이 튜토리얼이 에이전트의 세계를 탐험하는 출발점이 되어 대규모 언어 모델의 "사용자"에서 에이전트 시스템의 "구축자"로 전환할 수 있기를 바랍니다.

## 독자를 위한 제안

환영합니다, 미래 지능형 시스템 빌더! 이 흥미진진한 여정을 시작하기 전에 몇 가지 작은 제안을 드리고 싶습니다.

이 프로젝트를 읽기 전에 다음 사항을 확인하시기 바랍니다.

- 기본적인 Python 프로그래밍 기술을 보유하고 있습니다.

- 대규모 언어 모델에 대한 기본 개념을 이해하고 있습니다(예: LLM API를 얻는 방법을 알고 있음).

- 안심하세요. 알고리즘이나 모델 교육에 대한 깊은 배경 지식이 필요하지 않습니다. 이 프로젝트는 적용 및 건설에 중점을 둡니다.

이 프로젝트는 5개 부분으로 나누어 기본 사항을 실습하고 단계별로 진행합니다.

**1부(기초)**: 인공 지능 및 LLM에 대한 핵심 지식의 기초를 마련하여 에이전트 출현 배경에 대한 거시적 이해를 제공합니다.

**파트 2(단일 에이전트)**: 실습이 시작되는 곳입니다. 우리는 내부 "정신적" 구조를 깊이 이해하면서 완전히 기능하는 단일 에이전트를 처음부터 구축하도록 안내할 것입니다.

**파트 3(고급)**: 여기서 에이전트는 생각하고, 메모리와 도구를 보유하고, 에이전트 간 통신 프로토콜을 마스터하는 방법을 '학습'하여 궁극적으로 평가 폐쇄 루프를 완료합니다.

**4부(실습)**: 프로젝트의 핵심 가치가 여기에 있습니다. 신중하게 설계된 일련의 포괄적인 사례를 통해 배운 모든 지식을 통합하여 실제로 진정한 금을 단련하게 됩니다.

**파트 5(Outlook)**: 여정의 끝은 새로운 시작입니다. 귀하는 학습 여정의 완벽한 결론을 도출하는 "졸업 프로젝트"를 직접 만들게 됩니다.

"종이에서 배운 것은 피상적입니다. 진정으로 이해하려면 연습이 필요합니다." 최고의 학습 효과를 얻기 위해 프로젝트의 `code` 폴더에 모든 지원 코드를 제공합니다. 이론과 실습을 결합하는 것이 좋습니다. 프로젝트에 제공된 모든 코드를 직접 실행하고 디버그하고 수정하시기 바랍니다. 배운 내용을 관심 있는 실제 시나리오에 적용해 보시기 바랍니다. 이것이 학습의 궁극적인 목적입니다.

마지막으로, 오픈 소스 프로젝트로서 여러분의 참여와 기여를 진심으로 환영합니다. 문제가 발생하면 커뮤니티에 질문을 할 수 있습니다. 새로운 아이디어나 발견이 있으면 언제든지 프로젝트 공동 구축에 참여할 수 있습니다.

Hello-Agents를 읽어주셔서 감사합니다. 행복한 학습과 무한한 탐험을 기원합니다!
